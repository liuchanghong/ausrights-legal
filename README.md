# AusRights Legal Site

This folder contains the public legal pages for AusRights and is ready to be
published as a GitHub Pages site.

The public support email used by the policy is `liuchanghong@outlook.com`.

## Recommended GitHub Pages Setup

1. Create a public GitHub repository named `ausrights-legal`.
2. Upload the contents of this `LegalSite` folder to the root of that repository.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/ (root)` folder, then save.
6. After publication, the privacy policy URL will be:

   `https://liuchanghong.github.io/ausrights-legal/privacy/`

Use that public URL in App Store Connect and in the app's Settings privacy
policy link.

## App Store Connect Privacy Entry

For the current version of AusRights, select `Data Not Collected` in the App
Privacy questionnaire. The app stores only the selected display language on
the device and does not transmit it to you or to third parties.

## Optional Custom Domain

If you own the `auslaw.guide` domain currently referenced by the app, GitHub
Pages can serve this site at a custom domain such as:

`https://auslaw.guide/privacy/`

Configure the domain only after you control its DNS and have enabled HTTPS in
GitHub Pages.
